# Code of Conduct

This project is intended to be useful, direct, and respectful.

Expected behavior:

- Be clear and constructive.
- Assume good intent, but do not excuse harmful behavior.
- Keep discussions focused on the project.
- Respect maintainers' time and review scope.

Unacceptable behavior:

- Harassment, insults, threats, or personal attacks.
- Publishing private information.
- Repeated off-topic or bad-faith participation.

Maintainers may moderate issues, discussions, pull requests, and contributors to protect the project.
